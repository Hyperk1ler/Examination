## 第1章 绪论

### 1.1 知识框架

```
人类视觉基础 → 计算机视觉定义与目标 → 发展四阶段 → 经典理论 → 应用与挑战
```

### 1.2 人类视觉系统

- 人类约 **80%** 的信息通过视觉系统获取
- **信号路径**：视网膜感光细胞 → 神经脉冲 → 视神经 → **初级视觉皮层(V1区，枕叶)** → 次级视觉皮层(V2/V3) → 视觉联合区(**V4**处理颜色，**V5**处理运动) → **颞叶**(面孔/物体识别)和**顶叶**(空间定位/运动)

### 1.3 计算机视觉的定义与研究目标

| 维度 | 内容 |
|------|------|
| **定义** | 使计算机从图像/视频中"看"和"理解"现实世界的科学 |
| **输入** | 图像(images)或视频(video) |
| **输出** | 对象的恢复及对对象信息的使用 |
| **三大处理层次** | ①视觉信息获取(采样、量化、表示) → ②视觉信息处理(预处理、滤波、变换、分割、特征检测) → ③视觉信息理解(三维恢复重建、运动信息提取) |
| **研究目标** | 使机器像人那样通过视觉观察和理解世界，具有自主适应环境的能力 |

> 💡 **核心理解**：计算机视觉 = 图像处理 + 模式识别 + 三维理解，不只是简单的图像分类。

### 1.4 计算机视觉发展四阶段

| 阶段 | 时间 | 特点 | 代表性成果 |
|------|------|------|-----------|
| **启蒙阶段** | 1960s-1980s | 基础理论探索与建立 | Larry Roberts提出从二维图像提取三维形状 |
| **重构主义阶段** | 1980s-2000s | 传统算法快速发展 | Harris角点检测(1988)、SIFT算法(1999)、David Lowe基于知识的视觉 |
| **分类主义阶段** | 2000s-2010s | 技术成熟与应用探索 | Viola-Jones人脸检测、SVM/随机森林、HOG、LBP |
| **大数据大模型阶段** | 2010s-至今 | 深度学习爆发，大模型兴起 | Faster R-CNN、YOLO、GAN、BERT/GPT/CLIP |

> 💡 发展主线：几何三维重建 → 手工特征+分类器 → 深度学习端到端 → 大模型多模态

### 1.5 经典计算机视觉理论

#### Marr计算视觉理论
- **提出者**：David Marr (1945-1980)，1982年出版《视觉》
- **三个层次**：
  1. **计算理论** — 视觉系统要做什么
  2. **表达与算法** — 如何表示和计算
  3. **算法实现** — 硬件/软件实现
- **视觉信息处理三阶段**：
  1. **低层视觉**：从二维图像提取"要素图"（边缘、直线、曲线、纹理等基本元素）
  2. **中层视觉**：形成 **2.5维描述**（以观察者为中心的坐标系，部分的三维信息）
  3. **高层视觉**：构建完整三维描述（以物体为中心的坐标系）
- **局限性**：假定视觉是被动响应；认为物体表达就是三维几何形状

#### 主动视觉与多视几何
- **主动视觉**(Bajcsy)：强调视觉过程应融入人与环境的交互，核心研究"视觉注意力"机制
- **多视几何**：研究两视图间的对极几何(基础矩阵F)、三视图间的三焦张量、多视图间的单应约束

#### 基于学习的视觉理论
- **流形学习**：从图像表达中学习内在流形表达的非线性优化
- **深度学习**：依赖大规模数据集(ImageNet等)，以VGG、AlexNet、ResNet为代表

### 1.6 应用领域与挑战

- **应用领域**：安全监控、工业检测、军事国防、遥感气象、医学诊断、文化教育、智能交通、娱乐影视
- **面临挑战**：人类视觉机理尚不完全清楚；视觉感知技术有待变革；多模态融合需深入探索；复杂开放场景的模型泛化性亟待提高

## 第2章 图像表示和处理

### 2.1 知识框架

```
图像数字化(采样+量化) → 图像基本性质 → 数学基础(卷积+傅里叶变换) → 图像处理(像素变换+空域滤波+频域滤波)
```

### 2.2 图像表示：采样与量化

| 概念 | 说明 |
|------|------|
| **采样(Sampling)** | 将连续图像函数 $f(x,y)$ 离散化为 M×N 矩阵，采样间隔 Δx、Δy 决定分辨率 |
| **Shannon采样定理** | 采样间隔应 ≤ 图像中最小细节尺寸的 **一半** |
| **量化(Quantization)** | 将每个样本值量化为整数，b位表示 → 亮度阶 $k = 2^b$，通常每像素 **8比特**(256灰度级) |
| **精度原则** | M、N、K越大，连续图像函数的近似性越好 |

### 2.3 图像基本性质

#### 距离度量
- **欧氏距离**：$D_E = √[(x-u)² + (y-v)²]$
- **城区距离(D4)**：$D_4 = |x-u| + |y-v|$
- **棋盘距离(D8)**：$D_8 = max(|x-u|, |y-v|)$

#### 邻接性与区域
- **4-邻接**：两像素距离 D4=1
- **8-邻接**：两像素距离 D8=1
- **区域**：彼此邻接的像素集合；连通关系具有自反性、对称性和传递性
- **简单连通** vs **复连通**（区域中有孔）

#### 边缘 vs 边界 ⭐重要区分
| 概念 | 定义 | 性质 |
|------|------|------|
| **边缘(edge)** | 灰度不连续/剧烈变化的局部性质 | 矢量，有大小和方向 |
| **边界(border)** | 区域相关的全局概念 | 区域R的边界 = 每个点具有R外的邻接点 |

> 💡 边缘是"局部灰度变化"，边界是"区域轮廓"。寻找边界常通过连接显著边缘来实现。

#### 直方图
- 给出图像中各亮度值出现的概率分布，大小为 L 的一维数组

#### 噪声
| 类型 | 特点 |
|------|------|
| **加性噪声** | 与图像信号无关，$f' = f + n$ |
| **乘性噪声** | 幅值与信号幅值相关，$f' = f × n$ |
| **白噪声** | 常量的功率谱，强度不随频率衰减 |
| **高斯噪声** | 服从正态分布 |
| **信噪比(SNR)** | $SNR = S²/N²$，值越大图像品质越好 |

### 2.4 图像处理数学基础

#### 卷积
- 二维函数 f 和 h 的卷积：$g(x,y) = f(x,y) * h(x,y) = ∫∫ f(a,b)·h(x-a, y-b) da db$
- 在数字图像上使用离散卷积

#### 傅里叶变换
- 将图像从**空域**变换到**频域**
- $F(u,v)$ 通常为复数：
  - **傅里叶频谱**：$|F(u,v)| = √[R²(u,v) + I²(u,v)]$ — 反映各频率成分的能量分布
  - **相位角**：$φ(u,v) = arctan[I(u,v)/R(u,v)]$ — 携带结构信息

### 2.5 图像处理

#### 像素亮度变换
- **底片变换**、**分段线性变换**、**二值化**

#### 直方图均衡化 ⭐核心算法
- **目标**：创建在整个亮度范围内均匀分布的图像
- **核心步骤**：
  1. 形成直方图 H
  2. 归一化累加得到累计直方图 Hc
  3. 变换 $T = p₀ + (p_k - p₀)·Hc$
  4. 重新扫描图像应用 T
- **效果**：增强直方图极大值附近的对比度，减小极小值附近的对比度

#### 空域滤波

**线性滤波**：输出像素 = 输入像素邻域 Ω 的亮度线性组合

```
g(m,n) = ΣΣ f(m-i, n-j) · h(i,j)    ( (i,j)∈Ω )
```

其中 h 称为滤波器/卷积掩膜/核/窗口。

**图像平滑（抑制噪声 = 抑制高频）**：

| 方法 | 原理 | 特点 |
|------|------|------|
| **平均法** | 邻域像素均值，噪声标准差降低 1/√n | 会模糊边缘 |
| **限制数据有效性平均** | 仅用灰度在合法范围内的像素 | 避免模糊 |
| **反梯度平均** | 根据反梯度计算权重 | 保持边缘 |
| **旋转掩膜平均** | 搜索一致性最好的区域进行平均 | 用亮度散布 σ² 度量一致性 |
| **中值滤波** | 用邻域亮度中值代替当前像素 | 对**椒盐噪声**特别有效 |

> 💡 平滑的关键矛盾：**抑制噪声 vs 保持边缘**。中值滤波和旋转掩膜是两种边缘保持方案。

#### 频域滤波

- **流程**：$f(x,y)$ → 傅里叶变换 → $F(u,v)$ → 滤波 $H(u,v)$ → $G(u,v)=H(u,v)·F(u,v)$ → 逆傅里叶变换 → $g(x,y)$

| 滤波类型 | 作用 | 说明 |
|----------|------|------|
| **低通滤波** | 保留低频（平滑区域），抑制高频（边缘/噪声） | 理想低通有振铃效应不可用；Butterworth效果较好；指数低通无振铃；梯形低通较常用 |
| **高通滤波** | 抑制低频，增强高频 | 突出边缘和细节，实现锐化 |
| **带阻/带通滤波** | 阻止/允许特定频段 | 用于去除周期性噪声（如正弦噪声） |
| **同态滤波** | 分离照射分量（低频）和反射分量（高频），分别处理后合成 | 能消除不均匀照度且不损失细节 |

> 💡 空域与频域技术可互相转换：频域低通 ↔ 空域平滑模板的傅里叶变换；频域高通 ↔ 空域锐化模板的傅里叶变换。

## 第3章 图像点的特征表示

### 3.1 知识框架

```
角点定义 → Harris角点检测 → SIFT尺度不变特征变换 → 深度学习特征点 → 图像配准
```

### 3.2 Harris 角点检测 ⭐

#### 角点定义
- 角点 = **两条边缘的交汇点**，局部区域内梯度方向发生显著变化
- 对旋转和视角变化具有稳定性

#### 核心思想
滑动窗口观察灰度变化：
- 各方向都无变化 → **无角点（平坦区域）**
- 某方向变化大但另一方向不变 → **边缘**
- 各方向变化都大 → **角点**

#### 算法三步
1. **统计局部窗口内灰度变化**：计算自相关矩阵 M
2. **计算角点响应函数**：$R = det(M) - k·[trace(M)]²$
3. **角点判定**：R > 阈值 且为局部极大值

### 3.3 SIFT 尺度不变特征变换 ⭐⭐核心算法

**提出者**：David Lowe (1999)

**四大不变性**：尺度不变性、旋转不变性、光照不变性、对遮挡鲁棒

#### 四步流程

| 步骤 | 内容 | 关键技术 |
|------|------|----------|
| **①尺度空间构建及极值点检测** | 高斯金字塔 → DoG差分 → **26邻域**(当前8+上下层各9)极值检测 | 多尺度保证尺度不变性 |
| **②关键点定位** | 三维二次函数拟合精确定位 → 剔除低对比度点 → 剔除边缘点（Hessian特征值比率判断） | 提高稳定性 |
| **③方向赋值** | 计算梯度幅值和方向 → 构建**36柱方向直方图**(每柱10°) → 峰值为主方向(>80%峰值为辅方向) | 实现旋转不变性 |
| **④描述子生成** | 16×16邻域 → 16个4×4子区域 → 每子区域8柱方向直方图(45°/柱) → **128维**特征向量 → 归一化 | 实现光照不变性 |

### 3.4 深度学习特征点方法

| 方法 | 特点 |
|------|------|
| **HardNet** | L2Net特征学习网络 + 困难样本SoftReg采样策略，侧重描述子学习 |
| **Key.Net** | 结合手工设计滤波器和可学习滤波器，多尺度金字塔(三级1.2倍降采样)，侧重关键点检测 |

### 3.5 图像配准

- **定义**：寻找空间变换将一幅图像映射到另一幅，使对应点一一对应
- **四步流程**：关键点检测 → 特征描述子提取 → 特征匹配 → 参数估计与图像变换（如单应矩阵）
- **常用特征**：角点(Harris)、SIFT、SURF等

## 第4章 图像的线特征表示

### 4.1 知识框架

```
边缘检测基础 → 一阶/二阶微分算子 → Canny边缘检测器 → Snake活动轮廓 → ASM/AAM → Hough变换
```

### 4.2 边缘检测基础

#### 边缘产生原因
物体边界、表面方向改变、不同颜色或材质、光照明暗变化

#### 边缘类型
- **阶梯状边缘**、**屋脊状边缘**、**线条状边缘**

#### 边缘检测三步骤
```
抑制噪声(低通滤波/平滑) → 边缘特征增强(高通滤波/锐化) → 边缘定位
```

> 💡 边缘检测是一个"**病态问题**"——噪声导致微分放大的误差。解决方案：**先平滑(滤波)再求导**。

#### 梯度算子原理
- **一阶导数**：极大值点 → 梯度幅值大 = 边缘强
- **二阶导数**：过零点（须有符号改变）
- **梯度方向** = 灰度变化最快的方向

### 4.3 一阶与二阶微分算子

| 算子 | 模板尺寸 | 特点 |
|------|----------|------|
| **Roberts** | 2×2 | 计算简单，但定位精度受限于模板尺寸 |
| **Prewitt** | 3×3 | 平滑+检测结合，x方向 $[-1,0,1;-1,0,1;-1,0,1]$，y方向 $[-1,-1,-1;0,0,0;1,1,1]$ |
| **Sobel** | 3×3 | 在Prewitt基础上给四邻域更大权重，去噪+增强边缘效果更好 |
| **Laplacian** | 3×3 | 二阶微分，标量输出，旋转不变性。两种常见模板：$[0,-1,0;-1,4,-1;0,-1,0]$ 和 $[-1,-1,-1;-1,8,-1;-1,-1,-1]$。对噪声极敏感，不单独使用 |
| **LoG(高斯拉普拉斯)** | 通常5×5 | 先高斯平滑再Laplacian，形状如**墨西哥帽(Mexican hat)**。通过过零点检测边缘，结果单像素宽但可能产生spaghetti效应 |

### 4.4 Canny 边缘检测器 ⭐⭐核心算法

**提出者**：J. Canny (1986)

**三条最优准则**：
1. **好的检测**：对边缘响应 > 噪声
2. **好的定位**：接近实际位置
3. **单一边缘响应**：在边缘附近只有一个极大值点

**算法四步**：

| 步骤 | 操作 | 结果 |
|------|------|------|
| **①计算图像梯度** | 用高斯函数的一阶导数做卷积，获得梯度幅值 M 和方向 θ | 边缘增强 |
| **②非极大值抑制(NMS)** | 仅在梯度方向上保留极大值点 | **单像素宽边缘** |
| **③双阈值提取** | T1(低阈值)→E1(高召回但多误检)，T2(高阈值,T2>>T1)→E2(更可靠) | 可靠边+候选边 |
| **④边缘连接** | 从E2可靠边出发，在E1中寻找延伸直至与另一条可靠边连接 | 连续完整边缘 |

**参数选择**：Gauss滤波尺度 σ，双阈值 $LOW = HIGH × 0.4$

> 💡 Canny的核心创新：NMS → 单像素边；双阈值+边缘连接 → 连续完整且低误检。

### 4.5 Snake 活动轮廓模型

**提出者**：Kass, Witkin, Terzopoulos (1987)

- **定义**：能量达到最小值的曲线，需要交互指定初始形状和位置
- **能量函数 = 内力 + 外力 + 外部约束**：

| 能量项 | 来源 | 作用 |
|--------|------|------|
| **内力** | 来自形状 | 一阶项控制**弹性**（连续性），二阶项控制**刚性**（光滑性）。若二阶系数设为0，该处可产生角点 |
| **外力** | 来自图像 | 基于线（吸引到特定灰度轮廓）或基于边缘（吸引到梯度大的强边缘） |
| **外部约束** | 用户指定 | 迫使snake离开错误的局部极小值 |

> 💡 Snake是变形模板：初始轮廓 → 内力保持光滑连续 + 外力驱动到图像特征 → 能量最小化收敛。

### 4.6 ASM/AAM 模型

| 模型 | 全称 | 核心思想 |
|------|------|----------|
| **ASM** | Active Shape Model（主动形状模型） | 基于点分布模型PDM，用关键特征点(landmarks)形成形状向量描述目标。Building(归一化对齐→PCA降维→建立局部特征) + Fitting(匹配到新图像) |
| **AAM** | Active Appearance Model（主动外观模型） | 在ASM基础上增加**纹理(外观)信息建模**，同时利用形状和外观匹配 |

### 4.7 Hough 变换 ⭐

#### 基本思想
图像空间 (x,y) 中一条直线上的所有点 → 在参数空间中对应交于一点的若干条曲线。在参数空间中找**聚类点**，即可确定图像空间中的直线参数。

#### ρ-θ 参数化
为避免斜截式的 k 趋向无穷（垂直线），采用法线式：
```
ρ = x·cosθ + y·sinθ
```
图像中过 (x₀,y₀) 的直线群在 ρ-θ 空间中对应一条**正弦曲线**。

#### 算法实现
1. 准备 ρ-θ 二维数组（累加器）
2. 对每个边缘点，计算其对应的 ρ-θ 轨迹并在数组中**累加投票**
3. 提取**高票数组元素**即为检测到的直线

> 💡 Hough变换的本质是"**证据累积投票**"——每个边缘点为所有可能经过它的直线"投票"，票数最高的直线即为真实边缘。对不连续边缘和噪声具有强鲁棒性。可扩展到圆、椭圆等曲线检测。

## 第5章 区域分割

### 5.1 知识框架

```
分割准则 → 传统方法(阈值/区域生长/分裂合并/分水岭) → 深度学习(FCN/U-Net/DeepLab系列/SAM)
```

### 5.2 图像分割定义与四大准则

- **目的**：将图像细分为符合事先定义准则的多个独立区域
- **四大准则**：
  1. **相似性** — 像素特征相似 → 同区域
  2. **连续性** — 空间相邻
  3. **紧凑性** — 非碎片化
  4. **唯一性** — 每像素只属于一个区域

### 5.3 传统分割方法

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **阈值分割** | 全局/局部/自动阈值，基于灰度分布 | 计算简单、直观 | 对光照和噪声敏感 |
| **区域生长法** | 从种子像素开始，将相邻相似像素逐步合并 | 直观灵活 | 对种子点和参数敏感 |
| **分裂合并法** | 四叉树结构，不一致则分裂为4个子区域，相邻一致则合并 | 自动化程度高，能处理复杂结构 | 计算复杂度高 |
| **分水岭算法** | 将梯度图像视作地形，模拟水位上升填充集水盆地，构建水坝=分割边界 | 产生连续分割边界，识别多区域 | 易过分割（需预处理/标记） |

### 5.4 FCN 全卷积分割网络

**提出者**：Jonathan Long等人 (2015)

- **核心改变**：取消CNN最后的全连接层 → 替换为**全卷积层**(保持空间维度) + **上采样层**(恢复原图尺寸)
- **上采样方法**：反池化(depooling)、反卷积(deconvolution)、插值上采样
- **不足**：结果不够精细（上采样模糊）、编码器池化导致信息丢失（难以恢复边界和小目标细节）

> 💡 FCN的关键洞察：将分类网络改造为像素级预测网络，原理是全卷积 + 上采样。

### 5.5 U-Net 分割网络 ⭐

**提出**：2015年，用于生物医学图像分割

**U形架构**（编码器-解码器对称结构）：

| 模块 | 功能 |
|------|------|
| **编码器** | 逐层提取特征，逐步降采样 |
| **解码器** | 逐步恢复分辨率，上采样 |
| **跳跃连接** | 将编码器特征图与解码器对应层**通道拼接**（vs FCN的像素值相加），保留细节 |
| **最后卷积层** | 生成分割图 |

**优势**：小样本表现卓越，特别适合医学图像（肿瘤识别、器官定量、细胞计数）

> 💡 U-Net成功的三要素：对称结构 + 跳跃连接(保留细节) + 通道拼接融合(避免信息损失)。

### 5.6 DeepLab 系列 (V1–V3+)

| 版本 | 核心改进 | mIoU (PASCAL VOC 2012) |
|------|----------|------------------------|
| **DeepLab-V1** | 全卷积网络 + Dense CRF后处理 + **空洞卷积**(扩大感受野) | 71.6% |
| **DeepLab-V2** | 引入**ASPP**(空洞空间金字塔池化)，多尺度特征；VGG16→ResNet-101 | 79.7% |
| **DeepLab-V3** | 改进ASPP(高空洞率卷积→1×1卷积+图像池化层)，移除CRF实现端到端 | 85.7% |
| **DeepLab-V3+** | 编码-解码架构 + 改进的Xception主干(深度可分离卷积) | **89.0%** |

> 💡 DeepLab演进主线：空洞卷积 → ASPP多尺度 → 端到端 → 编码解码+深度可分离卷积。

### 5.7 SAM 分割大模型

**SAM (Segment Anything Model)**

- **核心思想**：通过**提示驱动**（目标框、点、掩码）指定分割内容，而非固定类别的像素分类
- **三部分构成**：

| 组件 | 功能 |
|------|------|
| **图像编码器** | MAE预训练的ViT，处理高分辨率输入，对每张图只运行一次 |
| **提示编码器** | 将提示映射到特征空间 |
| **掩码解码器** | 融合图像编码+提示编码 → 生成分割掩码及置信度分数 |

- **训练规模**：1100万张图像上训练超过**10亿**掩码 → 强大零样本学习能力

## 第6章 纹理分析

### 6.1 知识框架

```
纹理概念 → 经典方法(灰度共生矩阵/Gabor小波) → 深度学习方法(T-CNN/PCANet)
```

### 6.2 纹理特征

- **纹理**：物体表面的触感或视觉感知，基本元素称为**纹理基元（纹元）**
- **三大视觉特征**：
  - **粗糙性** — 细腻 vs 粗糙
  - **方向性** — 水平/垂直/斜向
  - **周期性** — 元素空间重复
- **应用**：医学影像(病变检测)、遥感影像(地物分类)、工业检测(表面缺陷)

### 6.3 灰度共生矩阵 GLCM ⭐

**提出者**：Haralick等人 (1970s)

- **定义**：对 G 个灰度级的图像，给定位移矢量 $d=(dx,dy)$，GLCM 是一个 **G×G 矩阵**，元素 $Pd(i,j)$ 反映灰度为 i 和 j 的像素在相距 d 的位置上同时出现的次数/概率
- **方向 θ** 常取 0°、45°、90°、135°
- **统计度量**：能量、熵、对比度、同质性、相关性等

> 💡 GLCM统计的是"一对灰度值在特定方向和距离上共现"的分布，是**二阶统计量**。

### 6.4 Gabor 小波纹理分析

- **Gabor函数** = 正弦平面波调制的高斯核函数
- 二维Gabor滤波器是一个**带通滤波器**，对特定方向和频率敏感
- 模拟人类视觉皮层中**简单细胞的感受野**特性
- **纹理分析**：用一组不同主频和方向的Gabor核与图像卷积 → 获得每个频率邻域的响应 → 多通道纹理特征描述

### 6.5 深度学习纹理分析

| 方法 | 核心思想 | 特点 |
|------|----------|------|
| **T-CNN** | VGG19主干，用**Gram矩阵**(不同特征图的内积)描述纹理的空间位置无关统计量；纹理生成通过梯度下降最小化Gram矩阵的均方距离 | 说明纹理是"空间位置无关的统计特征" |
| **PCANet** | 三阶段：堆叠PCA滤波 → 二值哈希编码 → 直方图统计 | 对光照变化和遮挡具有惊人鲁棒性，甚至优于某些深度方法 |

## 第7章 摄像机成像模型

### 7.1 知识框架

```
成像原理(小孔+凸透镜) → 坐标系与齐次坐标 → 针孔成像模型(内参+外参+畸变) → 摄像机标定(DLT/张正友/一维标定物)
```

### 7.2 成像原理

| 方式 | 原理 | 特点 |
|------|------|------|
| **小孔成像** | 光通过小孔在后方平面形成倒立图像 | 小孔太小→通光量小/信噪比低；小孔太大→成像模糊 |
| **凸透镜成像** | 光线折射聚焦 | 平行于主光轴的光线经凸透镜汇聚于焦点，光心到焦点的距离为**焦距 f** |

### 7.3 坐标系与齐次坐标

#### 齐次坐标
- **定义**：用 n+1 维向量表示 n 维向量
- **两大优点**：
  1. 矩阵统一表示平移+旋转+缩放
  2. 可表示无穷远点（最后一维为0）

#### 四个坐标系

| 坐标系 | 表示 | 说明 |
|--------|------|------|
| **图像坐标系(像素)** | $(u, v)$ | 以像素为单位 |
| **图像坐标系(物理)** | $(x, y)$ | 以物理单位表示，每像素物理尺寸 dx, dy |
| **摄像机坐标系** | $(Xc, Yc, Zc)$ | 以摄像机光心为原点 |
| **世界坐标系** | $(Xw, Yw, Zw)$ | 三维世界中的绝对坐标系 |

### 7.4 针孔成像模型与摄像机参数 ⭐

#### 内参矩阵 K（5个参数）
包含焦距 $(fx, fy)$、主点坐标 $(cx, cy)$、倾斜因子 $s$
- 描述**摄像机坐标系 → 图像坐标系**的映射
- 内参回答"摄像机**如何**看"（与场景无关）

#### 外参矩阵 [R|t]
3×3 旋转矩阵 R（3个自由度）+ 3×1 平移向量 t
- 描述**世界坐标系 → 摄像机坐标系**的刚性变换
- 外参回答"摄像机**在哪里**看"（每次拍摄不同）

#### 投影方程
```
x = K [R|t] X
```
其中 x 为图像齐次坐标，X 为世界齐次坐标。

#### 相机畸变 ⭐

| 类型 | 产生原因 | 表现 |
|------|----------|------|
| **径向畸变** | 凸透镜影响，越靠近图像边缘越明显 | **桶形畸变**(放大率随距光轴距离增加而减小)和**枕形畸变**(相反) |
| **切向畸变** | 凸透镜与成像平面不严格平行 | 图像点沿切向偏移 |

### 7.5 摄像机标定方法

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **DLT直接线性变换** | 从三维点与图像点的投影关系建立线性方程组(≥6组对应点，不能共面)→求解P(3×4)→RQ分解得K和R | 精度高、计算简单 | 需要高精度3D标定物，灵活性不足 |
| **张正友棋盘格标定** ⭐ | 四步：①求解单应矩阵H(向量叉积+SVD最小二乘) → ②利用旋转矩阵正交性求K和外参(构造对称矩阵B=K⁻ᵀK⁻¹) → ③建立径向畸变模型求畸变系数(最小二乘) → ④最大似然优化(Levenberg-Marquardt全局优化) | 仅需打印棋盘格，操作简便，精度高 | 需确定模板点阵物理坐标 |
| **一维标定物法** | 一组共线点组成的线段，至少3点共线且已知两点距离 | 无遮挡问题，适合多摄像机系统 | 需≥6幅不同姿态图像 |

> 💡 张正友方法是介于传统标定和自标定之间的方法，既避免了传统方法设备要求高的缺点，又较自标定方法精度高。

## 第8章 三维视觉重建

### 8.1 知识框架

```
重建流程 → 多视几何(F/E矩阵) → SFM运动恢复结构 → MVS多目立体视觉 → 深度学习(MVSNet) → 主动技术(结构光/激光/光度立体)
```

### 8.2 三维重建流程

```
数据采集 → 图像预处理 → 摄像机标定 → 特征匹配与配准 → 深度估计与点云生成 → 点云处理 → 表面重建与纹理映射 → 优化与精细调整
```

**应用**：文化遗产保护、建筑设计与城市规划、医学影像与手术规划、VR/娱乐、工业制造

### 8.3 多视几何：极几何、基础矩阵与本质矩阵 ⭐

#### 极几何关系
三维点 P 在两幅图像上的投影点 p₁ 和 p₂ 满足：
```
p₂ᵀ F p₁ = 0
```
- **基础矩阵 F**：3×3，秩2，7自由度，包含内参信息（从像素到像素）
- **本质矩阵 E**：$E = K₂ᵀ F K₁$，仅含外参信息（归一化坐标之间）

#### 基础矩阵估计
- 每对匹配点提供一个约束方程
- **8点算法**求解
- 施加秩2约束：SVD后设最小奇异值为0

> 💡 F 包含内参信息（像素→像素），E 仅含外参信息（归一化坐标→归一化坐标）。

### 8.4 SFM 运动恢复结构 ⭐

**SFM = Structure from Motion**，同时恢复"摄像机在哪里"（运动）和"场景长什么样"（结构）

#### 两视角 SFM 流程
```
内参标定 → 寻找对应点 → 计算F → 从E分解得4种可能的R和t → 选择使三维点在两相机前方的那组 → 三角化获得三维坐标
```

#### 多视角 SFM
- 依次添加图像或融合重建结果
- **捆绑调整(Bundle Adjustment)**：通过最小化投影点与实际图像点的距离，全局优化投影矩阵和重建结果

### 8.5 MVS 多目立体视觉 ⭐

#### 视差与深度关系
```
视差 d = x_l - x_r
深度 Z = b·f / d
```
其中 b 为基线长度，f 为焦距。**视差越大 → 深度越小（物体越近）**。

#### 五步流程

| 步骤 | 内容 |
|------|------|
| **①极线矫正** | 使两图像光心水平对齐，将二维搜索简化为一维搜索 |
| **②图像预处理** | 处理光照不一致、噪声、镜面反射、遮挡等 |
| **③匹配代价计算** | Census、NCC等方法 |
| **④代价聚合** | 半全局/全局优化 |
| **⑤视差计算与优化** | 赢家通吃(WTA) + 亚像素精度优化 |

#### 多视角点云融合：ICP算法
- **ICP(Iterative Closest Point)**：寻找最优刚体变换 (R,t) 使源点云与目标点云最大程度重合

### 8.6 深度学习与主动重建技术

| 方法 | 类型 | 原理 |
|------|------|------|
| **MVSNet** | 被动（深度学习） | 可微单应变换 → 3D代价体 → 3D CNN正则化 → 回归深度图 (ECCV 2018) |
| **结构光** | 主动 | 投影仪投射已知模式 → 相机捕获调制图案 → 解码匹配 → 三角化。**相移法**通过N步相移条纹的相位值确定深度 |
| **激光扫描** | 主动 | 激光束照射目标 → 测量反射/回波时间或相位 → 获取距离 → 生成点云。应用于地理测绘、文化遗产、工业制造 |
| **光度立体** | 被动 | 固定视点、不同光照方向的多幅图像 → Lambertian反射模型 → 估计法向量 → 积分恢复深度。三个假设：正交投影 + 远处单一点光源 + 漫反射表面 |

## 第9章 运动分析

### 9.1 知识框架

```
运动分析基本问题 → 时间差分法 → 背景减除法(8种模型) → 光流法(HS/LK)
```

### 9.2 运动分析基本问题

- **核心问题**：估计运动前后相邻时刻两幅图像上对应点坐标之间的差值（**运动矢量**）
- **核心难点**：观察到的二维运动与真实三维运动不一致（投影导致歧义）
- **分类标准**：摄像机数目(单/多)、摄像机是否运动、目标数目(单/多)、目标类型(刚体/非刚体)
- **完整流程**：运动目标检测 → 跟踪 → 分类 → 行为理解

### 9.3 三种运动检测方法对比 ⭐

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **时间差分法** | 比较相邻帧差异，\|f(ti)-f(tj)\| > T | 鲁棒性好、运算量小、易于实现 | 对噪声敏感、运动实体内部易产生空洞、阈值T缺乏自适应性 |
| **背景减除法** | 背景建模 → 当前帧与背景模型比较 → 差异像素为前景 | 可获取较完整目标区域 | 背景模型建立、保持和更新是关键挑战 |
| **光流法** | 分析光流场，计算运动场后对场景分割 | 能提取目标完整运动信息 | 计算复杂度高、抗噪性能差 |

### 9.4 时间差分法

#### 差图像
```
d(x,y) = |f(x,y,ti) - f(x,y,tj)| > T  → 运动目标像素
```

#### 累积差图像 ADI
| 功能 | 说明 |
|------|------|
| ①速度估计 | 梯度方向 = 速度方向，梯度大小 ∝ 速度 |
| ②目标尺寸 | 像素个数帮助确定目标尺寸和移动距离 |
| ③运动历史 | 包含全部运动历史，有助于检测慢运动和尺寸小的目标 |

### 9.5 背景减除法 ⭐

#### 八种背景模型

| 模型 | 核心思想 | 适用场景 |
|------|----------|----------|
| **均值模型** | 前K帧中>50%时间呈现背景值的像素 | 简单场景 |
| **自适应模型** | $B_k = α·I_{k-1} + (1-α)·B_{k-1}$ | 光照缓慢变化 |
| **双背景模型** | 快慢两个更新速度 | 平衡时效与稳定 |
| **单高斯模型** | 每像素灰度服从正态分布 $N(μ, σ²)$ | 简单单模态背景 |
| **混合高斯模型** ⭐ | 多个高斯函数描述多模态背景 | 树叶晃动、水面波纹、雨雪天气 |
| **纹理模型(LBP)** | LBP直方图描述，对亮度变化不敏感 | 光照变化场景 |
| **ViBe** | 单帧初始化 + 随机更新 + 颜色空间距离判断 | 快速初始化需求 |
| **CodeBook** | 每像素创建CodeWords → 匹配/更新/删除 | 周期性背景变化 |

#### 三大挑战
1. 未充分利用相邻像素相关性 → **形态学滤波**后处理
2. 背景更新速度与目标运动速度不匹配 → 虚影或漏检
3. 树叶摇动和阴影被误检为前景 → 多摄像机深度信息辅助

### 9.6 光流法 ⭐⭐

#### 光流约束方程
基于**灰度不变性假设**（$dI/dt = 0$）：
```
I_x·u + I_y·v + I_t = 0
```
一个方程，两个未知数 (u,v) → **孔径问题(Aperture Problem)**，需要额外约束。

#### HS 算法 (Horn & Schunck, 1981)
- 引入**全局平滑性约束**：假设光流在整个图像上光滑变化（速度变化率为0）
- 通过变分法最小化数据项 + 平滑项的能量函数

#### LK 算法 (Lucas & Kanade, 1981)
- 引入**局部平滑性约束**：假设小空间邻域内运动矢量恒定
- 使用加权最小二乘法求解：$v = (AᵀW²A)⁻¹AᵀW²b$

| 算法 | 约束类型 | 特点 |
|------|----------|------|
| **HS** | 全局平滑 | 产生稠密光流 |
| **LK** | 局部平滑 | 产生稀疏光流（在特征点处） |

> 💡 改进方向：金字塔光流（多尺度）、区域光流法、特征光流法。

## 第10章 计算机视觉应用

### 10.1 知识框架

```
视觉分类(ResNet/ViT) → 目标检测(Faster R-CNN/YOLO) → 目标跟踪(MeanShift/FCNT/SiamMOT)
```

### 10.2 视觉分类 ⭐

#### 定义与评价指标

| 指标 | 含义 |
|------|------|
| **正确率(Accuracy)** | 分对样本数 / 全部样本数 |
| **Top-1指标** | 分类器第一个预测结果的正确率 |
| **Top-5指标** | 分类器前5个预测结果中有一个正确即算对 |

**传统方法 vs 深度方法**：传统采用人工特征提取+分类器(SVM等)；深度学习端到端学习特征+分类。

#### ResNet（残差网络）⭐
- **提出**：微软实验室 (2015)，ImageNet竞赛分类+检测第一名，COCO检测+分割第一名
- **核心创新**：**残差结构**——显式地让网络层去拟合残差映射，而非直接将堆叠层拟合所需的映射
- **解决问题**：深层网络的梯度消失/爆炸、退化(degradation)问题
- **三大亮点**：超深网络结构(突破1000层)、residual模块（主分支+捷径分支）、Batch Normalization(代替Dropout)
- **五个版本**：ResNet18/34（浅层，使用Basic Block）；ResNet50/101/152（深层，使用Bottleneck Block）

#### Vision Transformer (ViT)

**核心思想**：将图像当作文本序列处理

**处理流程**：
```
[224,224,3] → 切分为16×16 patch → [14,14,768] → [196,768]
→ 拼接[class]token → [197,768]
→ 添加Position Embedding
→ 输入Transformer Encoder(L个多头注意力+MLP交替块)
→ 分类头输出
```

#### 常用数据集
- **CIFAR-10**：10类，60000张32×32彩色图像
- **ImageNet**：1400万+图像，覆盖各类物体和场景

### 10.3 目标检测 ⭐

#### 传统方法 vs 深度方法

| 范式 | 代表方法 | 原理 | 特点 |
|------|----------|------|------|
| **传统** | 滑动窗口法 | 不同尺度和位置的窗口滑过图像→对每个窗口分类 | 实现简单但计算复杂度高 |
| **两阶段(深度学习)** | Faster R-CNN | ①RPN生成候选区域→②ROI Pooling→③分类+回归→④NMS去重 | 精度高但速度慢 |
| **一阶段(深度学习)** | YOLO、SSD | 将检测视为回归问题，直接从整图预测边界框+类别概率 | 速度快但早期精度偏低 |

#### Faster R-CNN
- **核心思想**：将目标检测分解为两个子任务——候选框生成(RPN) + 目标检测
- **流程**：输入图像 → CNN提取特征图 → RPN候选区域 → ROI Pooling → 全连接层(分类+回归) → NMS

#### YOLO ⭐

| 版本 | 核心改进 |
|------|----------|
| **YOLO v1** | 图像分**7×7网格** → 每格预测**2个边界框**和C类概率 → NMS去重。CVPR 2016 |
| **YOLO v3** | 多尺度预测 + 更强backbone，mAP-50=**57.9**(608分辨率)仅需**51ms**，速度远超两阶段方法 |

> 💡 检测核心矛盾：精度 vs 速度。两阶段(R-CNN系)偏精度，一阶段(YOLO系)偏速度。RetinaNet用Focal Loss缩小两者差距。

### 10.4 目标跟踪

#### 基本概念
- **目标跟踪**：利用视频/图像序列上下文信息，对目标外观和运动建模，预测运动状态并标定位置
- **跟踪难点**：形态变化、尺度变化、遮挡与消失、图像模糊

#### 三种跟踪范式（多目标跟踪 MOT）
| 范式 | 含义 |
|------|------|
| **SDE** | 分离检测与特征提取 |
| **JDE** | 联合检测与特征提取 |
| **JDT** | 联合检测与跟踪 |

#### 代表性算法

| 算法 | 核心思想 | 来源 |
|------|----------|------|
| **MeanShift** | 计算目标区域和候选区域的特征概率 → 相似函数度量 → MeanShift向量迭代 → 收敛到真实位置 | 经典 |
| **FCNT** | 联合VGG的conv4-3（局部判别特征→SNet）和conv5-3（语义特征→GNet），两个网络独立预测 | ICCV 2015 |
| **SiamMOT** | 基于孪生网络的多目标跟踪，隐式学习运动建模和关联 | CVPR 2021 |

**常用数据集**：MOT（多目标行人/车辆）、KITTI（自动驾驶）、VOT（单目标挑战）

