## 模块二：知识表示（第2章）— 作业计算题

### 2.1 知识表示方法
| 方法 | 说明 |
|:----:|------|
| **产生式规则** | IF 前提 THEN 结论/动作。系统=规则库+综合数据库+推理机 |
| **状态空间法** | (S, O, S₀, G) — 状态集合+操作符+初始+目标 |
| **一阶谓词逻辑** | 适合表示结构化知识，但推理效率低 |
| **RDF三元组** | (实体, 关系, 实体) 或 (实体, 属性, 属性值) |
| **知识图谱** | 图结构知识库，存储方案：属性型(实体,属性,属性值) + 关系型(实体1,关系,实体2) |

### 2.2 产生式系统
**三部分**：**规则库**(产生式集合) + **综合数据库**(事实库，存放当前信息) + **控制系统/推理机**(控制+推理程序)

### 2.3 状态空间表示法（计算题重点 ⭐）
**四元组**：**(S, O, S₀, G)**
- S：状态集合
- O：操作符集合
- S₀：初始状态
- G：目标状态

**解题步骤**：
1. 定义状态表示（如传教士问题：(m, c, b)）
2. 写出初始状态和目标状态
3. 列出所有安全操作（操作要满足约束条件）
4. 用搜索算法求解

## 模块三：搜索策略（第3章）— 作业计算题 ⭐

### 3.1 盲目搜索
| 算法 | 数据结构 | 完备性 | 最优性 | 特点 |
|:----:|:--------:|:-----:|:-----:|------|
| **DFS** | LIFO栈(Open表) | 有限完备，无限不完备 | ❌ | 省空间，可能陷入深渊 |
| **BFS** | FIFO队列(Open表) | ✅ 完备 | ✅ 每步代价相等时 | 占空间大 |

### 3.2 启发式搜索 ⭐
**A* 算法评估函数**：
$$f(n) = g(n) + h(n)$$
- $g(n)$：从起点到节点n的实际代价
- $h(n)$：从n到目标的最小估计代价（启发函数）
- **A* 条件**：$h(n) \leq h^*(n)$（h(n)是h*(n)的下界）→ 完备且最优

**A 算法 vs A* 算法**：
- A算法：对h(n)无限制 → 不完备、非最优
- A\*算法：h(n) ≤ h\*(n) → **完备且最优** ⭐

### 3.3 八数码问题（作业重点 ⭐）
**启发函数设计**（两种）：
1. **h₁(n) = 错位数码牌个数**（空格不计）
2. **h₂(n) = 曼哈顿距离之和**（各错位牌距目标位置横纵坐标差之和）

证明满足A\*条件：至少需要h(n)步才能归位 → h(n) ≤ h\*(n)

### 3.4 局部搜索
| 算法 | 核心 | 特点 |
|:----:|------|:----:|
| **爬山法** | 每步最佳邻接节点替换当前 | 不完备，易陷局部最大/高原/山脊 |
| **随机重启爬山法** | 多次随机初始点重启 | 以接近1的概率完备 |
| **模拟退火** | 允许下山，逐步降低概率 | 可能找到全局最优 |
| **遗传算法** | 选择+交叉+变异，有性繁殖 | 避免局部最优靠**变异** |

## 模块四：机器学习（第4章）— 概念题

### 4.1 ML三要素
**模型 + 损失函数 + 优化算法（梯度下降法）**

### 4.2 过拟合与欠拟合
| 状态 | 表现 | 解决 |
|:----:|------|:----:|
| **欠拟合** | 训练和预测效果都差 | 增加模型复杂度/特征 |
| **过拟合** | 训练极好，测试极差 | **Early-stopping / Dropout / 数据增强 / L1/L2正则化** |

### 4.3 监督学习算法
| 算法 | 用途 | 核心 |
|:----:|:----:|:------|
| **线性回归** | 回归（预测连续值） | y = wᵀx + b |
| **逻辑回归** | 分类（二分类） | Sigmoid输出[0,1]概率 |
| **KNN** | 分类/回归 | 计算距离→取K近邻→投票 |
| **SVM** | 分类 | 最大间隔分类超平面 |
| **决策树** | 分类/回归 | 树形结构，可解释性强 |
| **朴素贝叶斯** | 分类 | 贝叶斯定理 + 特征独立假设 |
| **集成学习** | 分类/回归 | 多个学习器组合（好而不同） |

### 4.4 过拟合的四种策略 ⭐
1. **Early-stopping**（早停）
2. **Dropout**（随机丢弃神经元）
3. **数据增强**（增加训练数据）
4. **权重正则化**（L1/L2）

### 4.5 无监督学习
| 算法 | 用途 | 核心 |
|:----:|:----:|:------|
| **K-Means** | 聚类 | 随机初始簇中心→分配→更新→迭代 |
| **PCA** | 降维 | 投影使方差最大，各主成分正交 |

### 4.6 弱监督学习
| 类型 | 核心 | 需人参与？ |
|:----:|------|:---------:|
| **主动学习** | 算法选最有价值的样本请求标注 | ✅ 需要 |
| **半监督学习** | 同时利用标注+未标注数据 | ❌ 不需要 |
| **迁移学习** | 源领域→目标领域知识迁移 | 不需额外标注 |
| **强化学习** | 智能体+环境+回报，试错学习 | 未必需要人 |

**迁移学习三种方法**：
1. 样本迁移
2. 特征迁移
3. **模型迁移（Fine-tune，最主流）**

### 4.7 评估方法
| 方法 | 原理 |
|:----:|------|
| **留出法(Hold-out)** | 按比例划分训练集/测试集 |
| **交叉验证(Cross-Validation)** | 分K份，K-1训练1测试，轮换K次取平均 |
| **自助法(Bootstrap)** | 有放回抽样N次，未抽到≈36.8%作测试集 |

### 4.8 其他概念
- **偏差(Bias)**：预测期望与真实值的偏离（拟合能力）
- **方差(Variance)**：不同训练集上的预测变动（稳定性）
- **归纳偏好**：算法在多个假设中做出选择的倾向（如奥卡姆剃刀偏好简单模型）
- **集成学习**："好而不同"——每个个体准确率好，多样性好（如随机森林）

## 模块五：人工神经网络（第5章）— 概念题 ⭐

### 5.1 感知机
- 1943年M-P模型 → 1958年Rosenblatt感知机
- **局限**：无法解决XOR等线性不可分问题（明斯基1969年指出）
- 单层感知机只能处理线性二分类

### 5.2 BP算法
**两个阶段**：
1. **正向传播**：输入→层层计算→输出
2. **反向传播**：误差→梯度下降→逐层更新权重

**BP不足之处**：
- 可能陷入局部极小
- 收敛速度慢
- 隐层节点数难以确定
- 对初始权重敏感

### 5.3 神经网络分类

**按连接范围**：
- **全连接神经网络**：每层神经元与上下层全连接
- **局部连接网络**：仅连接部分神经元（如CNN）

**按连接方式**：
- **前馈神经网络**：信息单向传播，无反馈
- **反馈神经网络**：有环路，如循环神经网络RNN

### 5.4 CNN三大核心操作 ⭐
1. **卷积**：提取局部特征
2. **池化**：降维 + 平移不变性（最大池化/平均池化）
3. **激活**：引入非线性（ReLU/Sigmoid/Tanh）

### 5.5 权重计算（作业重点 ⭐）
全连接神经网络：**连接权重总数 = 相邻两层神经元数乘积之和**

例：输入层5→隐层10→输出层2：
$$5 \times 10 + 10 \times 2 = 50 + 20 = 70$$

### 5.6 Sigmoid优缺点
| 优点 | 缺点 |
|:----:|:----:|
| 输出映射到(0,1)，适合概率解释 | **梯度消失**（两端饱和） |
| 平滑可微 | 输出非零中心（影响梯度更新） |
| 函数单调递增 | 计算量较大（指数运算） |

### 5.7 CNN典型结构
**输入层 → [卷积层+激活+池化层]×N → 全连接层 → 输出层**
- 卷积：提取特征
- 池化：降维+平移不变性
- **填充(padding)**：利用图像边界信息，不遗漏边缘
