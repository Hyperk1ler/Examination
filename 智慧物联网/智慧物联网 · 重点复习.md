# 智慧物联网 · 重点复习

> 根据考纲重点整理，覆盖 RFID、定位、传感器与传感网、短距离无线通信、远距离低功耗通信、安全与前沿七个模块。

## 一、RFID（射频识别）

### 1.1 定义
通过无线信号进行**标识和识别**的技术。

### 1.2 三大组件
| 组件      | 作用                       |
| ------- | ------------------------ |
| **阅读器** | 主动询问，发射电磁波 + 接收反射信号      |
| **天线**  | 传递射频信号                   |
| **标签**  | 含耦合元件 + 芯片 + 天线，携带唯一电子编码 |

**工作流程**：
```
阅读器发射电磁波 → 标签接收能量后激活 → 标签反射信息波 → 阅读器识别
```

### 1.3 RFID 标签分类

| 类型 | 电源 | 距离 | 特点 |
|:----:|:----:|:----:|------|
| **被动式**（无源） | 无电池 | <1m | 体积小、成本低、寿命长 |
| **主动式**（有源） | 自带电池 | 可达数十米 | 体积大、成本高、远距 |
| **半主动式** | 电池仅用于芯片运算 | — | 通信仍靠阅读器供电 |

**相较条形码的优势**：体积小、抗污性强、可重复读写、穿透性强、有循环冗余校验。

### 1.4 RFID 工作频率

|     频段      |       频率       | 耦合方式 |   距离   | 特点       | 应用      |
| :---------: | :------------: | :--: | :----: | -------- | ------- |
|  **低频 LF**  |   125/133kHz   | 电感耦合 |  <1m   | 穿透遮挡物佳   | 畜牧业、门禁  |
|  **高频 HF**  |    13.56MHz    | 电感耦合 |  <1m   | 可穿透多数遮挡物 | 图书管理、物流 |
| **超高频 UHF** | 433/860-960MHz | 电磁耦合 | 3-10m+ | 不能穿透水/金属 | 供应链、仓储  |

### 1.5 UHF 标签四区存储结构
- **保留区**：灭活口令和访问口令
- **EPC 区**：全球唯一电子产品编码
- **TID 区**：出厂固化的标签识别号
- **用户区**：自定义数据

### 1.6 RFID 定位
- **基于 RSS 定位**：LANDMARC 方案，精度 ≈ 1m
- **基于相位定位**：DAH 方案，可达毫米级
- **趋势**：相对定位、非绑定、交叉融合、深度学习

## 二、定位技术

### 2.1 位置信息三要素
**地理位置 + 时间 + 对象** → 提供因地制宜、见机行事、因人而异的服务。

### 2.2 四种基本定位方法

| 方法 | 原理 | 所需参考点 | 特点 |
|:----:|------|:---------:|------|
| **ToA**（基于距离） | 波速 × 时间 | ≥3 | 需时钟同步 |
| **TDoA**（基于距离差） | 双曲线方程联立 | ≥3 | 不需时钟同步 |
| **RSS 指纹** | 建指纹库 → 匹配定位 | — | 易实现，但环境变化影响大 |
| **AoA**（到达角） | 天线阵列测角 → 交会 | ≥2 | 精度高但需专门硬件 |

### 2.3 GPS 定位
- **空间段**：24 颗卫星
- **原理**：通过测量至少 4 颗卫星的 ToA（伪距），解算三维位置 + 时间
- **A-GPS**：利用蜂窝基站辅助快速锁定卫星，缩短首次定位时间至几秒

### 2.4 蜂窝基站定位

| 方式 | 原理 | 所需站数 | 特点 |
|:----:|------|:--------:|------|
| **COO** | 单站小区 ID | 1 | 简单快速但精度低（几百米级） |
| **ToA** | 信号传/传播时间 | ≥3 | 精度可达几十米 |
| **TDoA** | 到达时间差 | ≥3 | 不需基站同步 |
| **AoA** | 信号到达角 | ≥2 | 精度高但天线贵 |

**两个基站定位**：
- **AoA 方式**（两个站测角交汇即可）
- **ToA/TDoA** 理论上最少需 3 个站（二维定位需要至少两个距离差方程 / 三个距离方程）

### 2.5 室内定位技术
| 方法 | 原理 | 精度 | 局限 |
|:----:|------|:----:|------|
| **RFID 定位** | RSS / 相位 | ≈1m ~ 毫米级 | 需部署阅读器 |
| **声音信号定位** | TDoA / FMCW / 多普勒 | 较高 | 覆盖范围 2-3m，环境噪声干扰大 |
| **可见光定位** | LED 特征 | ≈十几 cm | 易被遮挡 |

## 三、传感器与传感网

### 3.1 传感器
- **定义**：感受物理量 → 输出电信号
- **组成**：**敏感元件** + **转换元件** + **电路**

### 3.2 传感网（Wireless Sensor Network）
传感网 = 物联网的感知层，将传感器 + 处理器 + 无线通信 + 电池集成一体。

| 对比项 | 普通传感器 | 传感网节点 |
|:------:|:----------:|:----------:|
| 功能 | 仅感知 | 感知 + 计算 + 通信 |
| 组成 | 敏感元件 + 电路 | 传感器 + MCU + 无线芯片 + 电池 |

**传感网通信特点**：
- **不用 TCP/IP**（节点资源极其有限）
- **自组网**（无固定网络结构）
- **多跳路由**（通过邻居节点接力传输）

### 3.3 ETX 路由协议
$$
ETX = \frac{1}{d_f \cdot d_r}
$$

| 参数 | 含义 |
|:----:|------|
| $d_f$ | 正向**收包率**（发送节点 → 接收节点） |
| $d_r$ | 反向 ACK **收包率**（接收节点 → 发送节点） |

选取 ETX 值**最小**的路径作为最优路由——即总重传次数最多的路径。考虑了链路质量的双向性。

## 四、短距离无线通信

### 4.1 Wi-Fi
| 项目 | 内容 |
|:----:|------|
| **标准** | IEEE 802.11 系列 |
| **频段** | 2.4 / 5 GHz |
| **调制** | DSSS、OFDM |
| **组网** | BSS（基站模式）：由 AP（接入点）+ 多个用户组成 |
| **AP 核心参数** | MAC 地址 + SSID |
| **优点** | 速率高（可达 Gbps 级） |
| **缺点** | 功耗大，不适合电池供电设备 |

### 4.2 蓝牙

| 项目 | 内容 |
|:----:|------|
| **标准** | IEEE 802.15.1 |
| **频段** | 2.4 GHz 跳频扩频 |
| **版本** | 蓝牙 4.0 BLE：延迟 6ms、128bit AES、峰值电流 <15mA |
| | 蓝牙 5：支持 2Mb/s 快速模式 + 400m 长距离模式 |
| **优点** | 低功耗（BLE）、成本低 |
| **缺点** | 速率和距离有限 |

### 4.3 Zigbee

|   项目    | 内容                     |
| :-----: | ---------------------- |
| **标准**  | IEEE 802.15.4          |
| **频段**  | 2.4 GHz                |
| **速率**  | 最高 250 kbit/s          |
| **特点**  | 低功耗、低复杂度、低速率、**自组织**网络 |
| **协议栈** | OSI 五层模型               |
| **应用**  | 传感器网络、工业监控             |

### 4.4 三种对比

| 维度 | Wi-Fi | 蓝牙 BLE | Zigbee |
|:----:|:-----:|:--------:|:------:|
| **标准** | 802.11 | 802.15.1 | 802.15.4 |
| **频段** | 2.4/5 GHz | 2.4 GHz | 2.4 GHz |
| **速率** | 百 Mbps ~ Gbps | 1~2 Mbps | 最高 250 kbps |
| **功耗** | 高 | 低 | **极低** |
| **组网** | 星型（AP 中心） | 星型/mesh | **mesh 自组网** |
| **典型应用** | 互联网接入 | 穿戴/健康 | 工业传感网 |

## 五、广域网与远距离通信

### 5.1 LPWAN 概述
| 特性 | 指标 |
|:----:|:----:|
| 距离 | 乡村可达 10 km |
| 速率 | Kb/s 级 |
| 成本 | < $5 |
| 电池寿命 | > 10 年 |

### 5.2 LoRa
**核心技术**：**线性扩频（CSS，Chirp Spread Spectrum）**
- 信号频率**充满整个带宽**，抗干扰能力极强
- 灵敏度达 **-148dBm**
- **扩频因子 SF** 控制 chirp 长度：**SF 越大 → 通信距离越远**，但速率越低

**三种工作模式**：
| 模式 | 特点 | 功耗 |
|:----:|------|:----:|
| **Class A** | 仅在上传后可短暂下载 | 最低 |
| **Class B** | 有预设的接收时隙 | 中 |
| **Class C** | 几乎连续接收窗口 | 最高 |

**LoRaWAN 安全**：
- AES-128 对称加密保护数据载荷
- 入网方式：ABP（手动激活）/ OTAA（无线配对激活）

### 5.3 NB-IoT

| 项目 | 指标 |
|:----:|:----:|
| 覆盖增强 | 比 GSM 提升 20-30 dB |
| 连接密度 | 10 万台 / km² |
| 电池寿命 | 5-10 年 |
| 芯片成本 | 低至 $1 |

**三种部署方式**：
- **带内部署**（In-band）
- **保护带部署**（Guard-band）
- **独立部署**（Stand-alone）

**省电模式**：
- PSM（省电模式）：最长 310h
- eDRX（增强型非连续接收）：延至 2.92h

### 5.4 LoRa vs NB-IoT

| 对比项 | LoRa | NB-IoT |
|:------:|:----:|:------:|
| **组网方式** | 私有化组网 | 蜂窝组网 |
| **频谱** | ISM 免费频段 | 授权频谱 |
| **部署主体** | 企业/个人 | 运营商 |
| **覆盖** | 好 | 更优（+20dB） |
| **成本** | 低 | 较高 |

### 5.5 毫米波通信
| 项目 | 内容 |
|:----:|------|
| **频率** | 30-300 GHz，波长 1-10 mm |
| **优势** | 频谱资源丰富（数 GHz 带宽）、高速率（802.11ad 达 8 Gbps）、易集成、高方向性 |
| **挑战** | 波束成形对齐困难、路径损耗大、空分多址、信道估计复杂 |

### 5.6 声音通信与可见光通信

**声音通信**：利用内置扬声器 + 麦克风
- 编码方式：**PIM / ASK / FSK / 扫频编码**
- BackDoor：利用 40-50 kHz 超声传输

**可见光通信（VLC）**：
- **发送端**：LED 灯 / 显示屏
- **接收端**：光电二极管 / 相机
- ChromaCode：120 Kb/s
- AirCode：超过 1 Mb/s

## 六、物联网安全

### 6.1 物联网安全威胁

| 威胁类型 | 说明 |
|:--------:|------|
| **DoS/DDoS** | 最常见，大量低功耗设备被控后发起攻击 |
| **物理攻击** | 直接破坏设备硬件 |
| **隐私攻击** | 窃听、被动监测用户行为 |

### 6.2 核心困境

> **物联网设备性能有限**（CPU、内存、电池），难以部署防火墙、杀毒软件、入侵检测等传统高级安全方案。传统安全 = 高性能计算 → 物联网做不到。

### 6.3 密码学四大目标

| 目标 | 手段 | 解决什么问题 |
|:----:|:----:|:-----------:|
| **机密性** | 加密 / 解密 | 防止信息被窃听 |
| **完整性** | 摘要算法（哈希） | 防止信息被篡改 |
| **真实性** | 消息认证码（MAC） | 确认信息来源合法 |
| **不可否认性** | 数字签名 | 发送者不能抵赖 |

### 6.4 LoRaWAN 安全
- 使用 **AES-128 对称加密**
- 入网方式：
  - **ABP**（Activation By Personalization）：手动配置
  - **OTAA**（Over-The-Air Activation）：无线配对激活，更安全
